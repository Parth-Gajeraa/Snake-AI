1.) Open MySQL Workbench2.) Click on + to add MySQL Connection            3.) Write snake_AI_2 as Connection Name and click on OK.4.) Give your Password of MySQL Workbench and click on OK to open snake_AI Connection5.) Once Connection gets open, click on Server to import a Data6.) Import a jdbc_snake_ai.sql file which you will find in zip7.) Click on New to create a new Schema and write down same name as of sql file: jdbc_snake_ai 8.) Click on Ok and import a Data9.) Once Data gets imported, close MySQL 10.) Copy the path of jdbc_snake_ai.sql 11.) Open Terminal and write the following: java -jar (path of jdbc file)/jar.jar (password of MySQL)In our case it was java -jar /Users/parthgajera/Desktop/jar.jar <Password>
